# AI Social Media Content Generator

A social media content generator built with Python and Streamlit.

## Features

- Business type input
- Platform selection
  - Instagram
  - LinkedIn
  - TikTok
- Tone selection
  - Professional
  - Funny
  - Luxury
  - Motivational

## Technologies

- Python
- Streamlit
- Git
- GitHub
